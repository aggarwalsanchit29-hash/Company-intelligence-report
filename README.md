# Company Intelligence Dashboard

AI-powered company intelligence reports across Banking, Healthcare, Retail & Energy sectors.

# Live Demo
https://company-intelligence-report-jqtom6yroyzztsdltez5ja.streamlit.app/

## Features
- Full Intelligence Reports
- KPI Snapshots with sector benchmarks
- Strategic Positioning Analysis
- Risk Assessments
- Download reports as Markdown

## Sectors Covered
- 🏦 Banking & Financial Services
- 🏥 Healthcare
- 🛒 Retail & FMCG
- ⚡ Energy & Utilities

## Setup
1. Add your `GROQ_API_KEY` to HuggingFace Space secrets
2. Run `streamlit run app.py`

## Tech Stack
- Python + Streamlit
- Groq (free LLM API)
- Zero database — fully stateless
